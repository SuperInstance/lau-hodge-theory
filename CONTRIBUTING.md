# Contributing

Thank you for your interest! This project is part of the [SuperInstance](https://github.com/SuperInstance) ecosystem.

## Quick Start

```bash
git clone https://github.com/SuperInstance/lau-hodge-theory.git
cd lau-hodge-theory
cargo build
cargo test
```

## Guidelines

- Follow existing code style (enforced by `cargo fmt` and `cargo clippy`)
- Add tests for new mathematical operations
- See [org-level contributing guide](https://github.com/SuperInstance/.github/blob/main/CONTRIBUTING.md) for full details

## Reporting Issues

Use [GitHub Issues](https://github.com/SuperInstance/lau-hodge-theory/issues) for bugs and feature requests.
