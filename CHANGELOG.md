# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-01-01

### Added
- HodgeStar operator for differential forms
- KnowledgeSpace with de Rham cohomology
- Wedge product, exterior derivative, and Laplacian
- Harmonic form computation
- Spectral sequence infrastructure
- Binomial coefficient and k-subset utilities
- Integration tests with nalgebra and approx
